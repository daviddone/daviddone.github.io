三方组件之高德地图

高德2D地图实现云搜索（周边地址选择）

获取SHA1与Package

Windows：依次在 eclipse 中打开 Window -> Preferances -> Android -> Build。
然后选择SHA1：18:F2:AA:4B:CD:FC:9D:BC:B8:64:0D:5D:4A:06:CA:E1:B9:EA:A5:A7
修改manifest文件：


aMapLocManager.requestLocationData(
LocationProviderProxy.AMapNetwork, 2000, 10, this);

AMap aMap;
private void addMarkersToMap(LatLng locationLaLng)
{
//添加每一帧图片。
markerOption = new MarkerOptions();//根据经纬度new LatLng(34.341568, 108.940174)
markerOption.position(locationLaLng);
markerOption.title(“西安市”).snippet(“西安市：34.341568, 108.940174”);
markerOption.draggable(false);// 地图拖拽
markerOption.icon(BitmapDescriptorFactory.fromResource(R.drawable.arrow));
aMap.addMarker(markerOption);
}

LatLng mTarget = aMap.getCameraPosition().target;

@Override
public void onCameraChange(CameraPosition cameraPosition) {
Log.e(“位置移动”,cameraPosition.toString());
}
@Override
public void onCameraChangeFinish(CameraPosition cameraPosition) {
Log.e(“位置移动结束”,cameraPosition.toString());
//获取拖动后的中央
LatLng mTarget = aMap.getCameraPosition().target;
}

aMap.setOnCameraChangeListener(this);// 对amap添加移动地图事件监听器

Android SDK 镜像更新

当遇到代理不稳定的情况，可以采用国内镜像资源更新Android SDK。不得不说，有时候镜像资源的速度还是非常给力的。这里推荐一下：国内镜像资源。

Step 1

打开Android SDK Manager 选择 Tools –> Options,然后在 Proxy Settings的Http Proxy Server和Http Proxy Port分别填上mirrors.neusoft.edu.cn和80，勾选上 [Force https://… sources to be fetched using http]，然后点击Close。

Step 2

然后选择Android SDK Manager下的Packages和Reload，这时候就可以更新了。
